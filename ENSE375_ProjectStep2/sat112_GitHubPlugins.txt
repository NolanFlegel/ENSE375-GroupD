Code Review: DeepSource

This plugin automatically helps identify and fix issues during code 
reviews. It boasts a <5% false-positive rate, faster code reviews 
(obviously), and an autofix feature to suggets fixes to identified issues.


Code Quality: Code Inspector

This plugin performs automatic code reviews and tracks quality metrics of
your code in order to show quality trends in the code over time. This data
is useful in managing quality practices of a developing team and discover
the best ways to improve quality and conistency.
